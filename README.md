# Bank

## Project setup
### Previous Requirements
* Linux / Mac OSX
* [Ruby 2.6.3](https://www.ruby-lang.org)
* [Rails 5.2.3](http://railsinstaller.org)
* [PostgreSQL 9.5](https://www.postgresql.org)
* [Bundler gem](https://github.com/bundler/bundler)

### 1. Install gems

    bundle install


### 2. Create database

    rails db:create

### 3. Migrate database

    rails db:migrate

### Running the server

    rails s

### Running the test switch

    rspec
